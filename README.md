# 🛒 Jordan Quality Store

## 📖 Descripción

**Jordan Quality Store** es una aplicación web desarrollada con **React** que simula una tienda en línea de sneakers de la marca Jordan.
El sistema permite a los usuarios explorar diferentes productos, agregarlos al carrito de compras y guardarlos en favoritos.

La aplicación está diseñada con una interfaz moderna utilizando **Material UI**, ofreciendo una experiencia visual atractiva y fácil de usar.

El proyecto fue desarrollado como parte del **Taller 3 de Desarrollo Web con React**, aplicando conceptos de componentes, rutas, estados globales y despliegue en la web.

---

# ⚡ Características Principales

* Visualización de diferentes **sneakers Jordan**.
* Sistema de **carrito de compras**.
* Sistema de **productos favoritos**.
* Barra de navegación con acceso a:

  * Inicio
  * Artículos
  * Ofertas
  * Mi cuenta
  * Favoritos
  * Carrito
* Uso de **Context API** para manejar estados globales.
* Interfaz moderna desarrollada con **Material UI**.
* Navegación entre páginas usando **React Router**.
* Botón en el Landing Page que redirige al **repositorio de GitHub**.

---

# 🎨 Interfaz Gráfica

La interfaz de la aplicación presenta un diseño moderno inspirado en tiendas deportivas online.

### Landing Page

En la página principal se muestra un **banner principal** con el mensaje:

> *Bienvenido a Jordan Quality Store*

Además incluye un botón que permite acceder directamente al **repositorio del proyecto en GitHub**.

### Barra de Navegación

El sistema cuenta con un **Header rojo** que incluye:

* Inicio
* Artículos
* Ofertas
* Mi Cuenta
* Favoritos
* Carrito
* Barra de búsqueda

### Sección de Productos

Los productos se muestran mediante **tarjetas (cards)** que contienen:

* Imagen del sneaker
* Nombre del producto
* Categoría
* Precio
* Botón **Comprar**
* Botón **Favorito**

# 🏗 Arquitectura del Proyecto

El proyecto está organizado utilizando una **arquitectura basada en componentes**, lo que permite una mejor organización del código.

## Estructura del proyecto

```bash
src
│
├── components
│   └── Header.jsx
│
├── pages
│   ├── Home.jsx
│   ├── Article.jsx
│   ├── Offers.jsx
│   └── Profile.jsx
│
├── contexts
│   ├── CartContext.jsx
│   └── FavoritesContext.jsx
│
├── App.jsx
└── main.jsx
```
